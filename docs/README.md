# Documents Folder

Project documents are currently being stored in the root `documents/` folder.

That folder can contain files such as:

- `mini-wind-energy-harvester-prd.docx`
- `dream-game-prd.docx`
- `sigfeed-prd.docx`

You can also store PDF versions there if you want the PRD to open in a browser tab more reliably.

After adding the files, update the matching links in the project pages under `projects/`.
