# Engineering Portfolio

A dark, electronics-inspired portfolio site for showcasing:

- featured engineering projects
- product requirement documents
- dedicated project detail pages with progress notes
- your profile as an electrical engineering student

## Files

- `index.html` contains the main landing page
- `styles.css` contains the visual system and circuit-like styling
- `script.js` handles the footer year
- `projects/` contains individual project pages
- `docs/` contains notes about document organization
- `documents/` stores downloadable Word or PDF documents

## Customize

Update these first: 

- your name in `index.html`
- your email and GitHub link in `index.html`
- project titles, descriptions, and links in `index.html`
- project detail content inside the `projects/` folder
- Word or PDF documents inside the `documents/` folder

## PRD Documents

Project pages are set up to link directly to PRD files in a new tab instead of using separate PRD web pages.

- place your `.docx` or `.pdf` files in `documents/`
- update each project page in `projects/` to link to the matching file
- prefer `.pdf` when you want the PRD to preview in the browser
- use `.docx` when PDF is not available yet, but note some browsers may download it instead of previewing it

## Preview

Open `index.html` in a browser to view the site locally.
